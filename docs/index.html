QuickHacks

A small macOS menu bar app for the everyday system tweaks that should not require hunting through settings or running Terminal commands.

One place for small Mac fixes.

QuickHacks lives in the menu bar. Open it when you need a quick switch, then get back to work.

Menu bar cleanup

Move noisy icons to the left of the separator and hide them when you do not need them.

Bluetooth Guard

Block devices that keep stealing audio or reconnecting at the wrong time.

Keep Awake

Prevent display sleep for a chosen duration, with a simple countdown.

Quick Toggles

Switch hidden files, desktop icons, Dock auto-hide, eject disks, and empty Trash.

Local and open source

No account, no sync service, no analytics. The code is public under the MIT license.

Native macOS

Built with SwiftUI and AppKit for macOS 14 and newer.

Download and install.

QuickHacks is distributed through GitHub Releases while notarization is being prepared.

  1. Download the latest ZIP from GitHub Releases.
  2. Unzip it and move QuickHacks.app to Applications.
  3. Open the app from Applications.
  4. If macOS warns that the developer cannot be verified, Control-click the app and choose Open.

Permissions explained.

macOS asks before apps can control sensitive system features. QuickHacks uses those permissions only for the features you choose.

Bluetooth

Used to list paired devices and disconnect devices you block.

Accessibility

Used to open other menu bar apps from the QuickHacks panel.

Finder Automation

Used for actions such as ejecting disks and emptying Trash.
