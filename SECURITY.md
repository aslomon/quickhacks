# Security Policy

QuickHacks is a small macOS utility that interacts with local system features
such as Bluetooth, Finder automation, and Accessibility.

## Reporting a Vulnerability

Please do not open a public issue for sensitive security reports.

Use GitHub's private vulnerability reporting if it is enabled for the
repository. If it is not enabled, open a minimal issue asking for a private
security contact without including exploit details.

## What to Include

- A short description of the issue.
- Steps to reproduce.
- The macOS version and QuickHacks version.
- Any relevant logs or screenshots that do not expose private data.

## Scope

Security reports are especially useful for:

- Unexpected command execution.
- Unsafe file operations.
- Permission bypasses.
- Privacy-sensitive data exposure.
- Behavior that could hide or control unrelated apps unexpectedly.
